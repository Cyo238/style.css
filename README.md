# IA Verificadora de Datos Históricos Coloniales

## Descripción del Proyecto

Este proyecto utiliza Inteligencia Artificial para analizar y verificar
información relacionada con la historia colonial.

La aplicación permite ingresar textos históricos y detectar posibles
referencias válidas sobre acontecimientos, personajes e instituciones
coloniales.

---

## Tecnologías Utilizadas

- HTML
- CSS
- JavaScript
- Python
- SQL
- JSON

---

## Funciones Principales

✅ Verificación de datos históricos  
✅ Detección de palabras clave coloniales  
✅ Organización de información histórica  
✅ Interfaz web interactiva  
✅ Base de datos histórica
